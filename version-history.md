# Version History

<!-- LATEST_VERSION: 1.53 -->

## v1.53 (Current)

- Unified modal system for loading, success, error, and warning states
- Refactored Generate Reports sidebar with collapsible settings cards
- Email Message editor upgraded to Trix rich text editor
- Email message now stored in UserProperties for faster load times
- Support menu refactored with fresh license on every load
- Dark mode and high-contrast mode improvements across all menus
- Selected course badge in Student Selection card header
- Unsaved changes indicator on Save button in Email Message
- Batch processing improvements for large classes
- Recent Reports and Delete Reports cards added

## v1.52

- Settings merged into main sidebar
- Single UserProperties config
- Folder cache TTL fix
- Recent reports and delete cards

## v1.51

- Google Classroom category-based grading support
- Show/hide category percent option
- Group by categories option

## v1.50

- Initial public release
- Generate and email student reports from Google Classroom
- Customizable email message with special codes
- Report options: grades, charts, comments, sorting
- Grade display format options
- Date filter for assignment range
- Columns and colors customization
