# Support

The Support menu shows your subscription details, version status, and provides a way to contact the Gdev Apps support team.

**Extensions → Grade Reports → Support**

---

## Version Status

Shows whether you have the latest version of Grade Reports installed.

- **Up to date** — You have the current version
- **Update Available** — A newer version exists. Click **Reinstall to Update** to get the latest version from the Google Workspace Marketplace. Reinstalling is safe — your data and settings are preserved.

---

## Subscription Details

| Field | Description |
|-------|-------------|
| **User** | Your Google account email |
| **License** | Your license type (see below) |
| **Version** | The currently installed version |
| **Expires** | License expiry date (not shown for Lifetime licenses) |
| **Daily Emails Remaining** | How many emails you can still send today via Gmail |

### License Types

| License | Description |
|---------|-------------|
| Free Trial | 15-day evaluation period |
| Annual | Individual license, renews yearly |
| Lifetime | Individual license, never expires |
| School Annual | Domain-wide license, renews yearly |
| School Lifetime | Domain-wide license, never expires |

### Refresh License Now

Click **Refresh license now** to fetch the latest license information from the server. This is useful if you've recently purchased or renewed a license and the details haven't updated yet.

---

## Upgrade License

If you're on a Free Trial or Annual license, an upgrade card will appear with options to purchase or upgrade your license.

- **Free Trial / Annual expired** → Purchase or renew at [gdev.app](https://gdev.app/products/grade-reports-for-google-classroom){:target="_blank"}
- **Annual (active)** → Upgrade to Lifetime for one-time payment access
- **School license** → Contact your school administrator for renewal

---

## Contact Support

Click **Email Support** to open a pre-filled Gmail compose window addressed to the Gdev Apps support team. The email automatically includes your ticket ID, user email, version, and license type to help the support team assist you faster.

{: .note }
> Support is available at [help.desk@gdev.app](mailto:help.desk@gdev.app)
