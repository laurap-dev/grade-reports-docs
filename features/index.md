# Features

Grade Reports provides three menus accessible from **Extensions → Grade Reports**:

- **[Generate Reports](generate-reports.md)** — The main menu for creating and emailing student reports
- **[Email Message](email-message.md)** — Customize the email sent to students and guardians
- **[Support](support.md)** — Subscription details and contact support
